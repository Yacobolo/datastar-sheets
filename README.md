# Datastar Quickstart

This is a simplified version of Nicholas Zanghi's datastart template: [Northstar](https://github.com/zangster300/northstar?tab=readme-ov-file)  
So go check that out and star his project.

# Setup

1. Install Dependencies

```shell
pnpm install
go mod tidy
```

2. Run

```shell
task live
```
